# OIBSIP_DataScience_Task3

## Car Price Prediction

### Objective

To predict the selling price of cars using machine learning regression techniques.

### Dataset

The dataset includes:

* Car Name
* Year
* Present Price
* Kms Driven
* Fuel Type
* Seller Type
* Transmission
* Owner

### Tools Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

### Steps Performed

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Data Visualization
5. Model Training
6. Price Prediction

### Results

The model successfully predicted car prices with satisfactory accuracy.

### Conclusion

Machine learning regression models can estimate vehicle prices based on historical features.
